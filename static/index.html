About this project:

With the buttons above, you can find the company name or domain name, given you have one of them. The data come from open-source sources throughout the internet, including Alexa's top 1M domains. The domains are only based off of Alexa's top 1M domains and will be updated occassionally. This project is built with a microservice architecture in mind.

This project utilizes:

  • Python
  • Bottle.py for micro web-framework
  • PostgreSQL for database
  • Nginx for web server
  • uWSGI for protocol

What I learned:

90% of the project is cleaning the data and open-source data is mostly not clean. When the data is good and normalized, serving the code is quite simple. The Bottle.py framework is great for quick deployments, especially with the single-file approach Bottle.py emphasizes. If I wanted to easily add authentication, rate-limits, and other ready-to-use extenstions, Flask.py would have been a better choice. However, for this project, Bottle with uWSGI does well it making it slim and flexible. I will eventually open-source this project and the data included. I want to add some more data to the project and spin up another microservice that will call this in the back-end.
